Html知识点回顾～噔噔噔

HTML之疑难杂症解决篇☺

1.meta标签

  • 优先使用IE最新版本和Chrome

    <meta http-equiv="X-UA-Compatible" content="IE=edge,chrome=1"/>

  • 浏览器内核控制。国内浏览器很多都是内双核（webkit和Trident）,webkit内核高速浏览，IE内核兼容网页和旧版网站。使用meta标签可以选择内核

    <meta name="renderer" content="webkit|ie-comp|ie-stand"/>

IE内核（兼容模式）:搜狗告诉浏览器，QQ浏览器
webkit内核（极速模式）:360极速浏览器，遨游浏览器

2.q标签/blockquote标签

这是这次回顾中新学到的两个标签。

  • q标签:短文本引用，样式为给一段文字加了双引号。
  • blockquote:长文本引用。

3.code标签/pre标签

  • code标签:在网页中显示计算机专业的编程代码，并且代码为一行时。
  • pre标签:插入多行代码。

4.列表

注意注意:以无序列表举例，ul为列表，li为列表项，必须成套使用。

  • caption 表格标题
  • summary属性 表格的摘要

5.表单

  • input输入框 当type=”password”时, 输入框为密码输入框。name 为文本框命名，以备后台程序ASP 、PHP使用。value 为文本输入框设置默认值。(一般起到提示作用)
  • radio单选框。
    注意注意:同一组的单选按钮，name取值一定要一致，这样同一组的单选按钮才可以起到单选的作用

  • label标签

    例子:

    <label for="haha">用户名</label>
<input type="text" name="" value="哈哈" id="haha">

    当用户选择该标签时，浏览器就会自动将焦点转到和标签相关的表单控件上。for属性可把 label 绑定到另外一个元素。请把 “for” 属性的值设置为相关元素的 id 属性的值

  • textarea文本域
    通过css中的resize:none可以去除文本框的变动

  • select下拉列表框

    <select multiple="multiple"></select>

    multiple属性可实现下拉框多选功能
